一、基本的数据类型

===list=== 列表
//--Asd
线性表

>>> names = ['Michael', 'Bob', 'Tracy']
>>> names
['Michael', 'Bob', 'Tracy']

>>> names = [1, 2, 4]
>>> names
[1, 2, 4]
>>> names.insert(1,1)
>>> names
[1, 1, 2, 4]
>>> 

===tuple=== 元组
//--Asd
线性表 常量

>>> classmates = ('Michael', 'Bob', 'Tracy')
>>> classmates
('Michael', 'Bob', 'Tracy')
>>> t = (1, 2)
>>> t
(1, 2)
>>> t1 = ()
>>> t1
()
>>> type(t1)
<class 'tuple'>
>>> t2 = (1)
>>> t2
1
>>> type(t2)
<class 'int'>
>>> t3 = (1, )
>>> t3
(1,)
>>> type(t3)
<class 'tuple'>


===dict=== 字典
//--Asd
key唯一，无序

>>> d = {'Michael':95, 'Bob':75, 'Tracy':85}
>>> d
{'Tracy': 85, 'Bob': 75, 'Michael': 95}
>>> d['Michael']
95
>>> d = {'a':12, 'b':23, 'c':34}
>>> d
{'b': 23, 'a': 12, 'c': 34}
>>> d['a']
12
>>> type(d)
<class 'dict'>
>>> d['d'] = 45
>>> d
{'b': 23, 'd': 45, 'a': 12, 'c': 34}
>>> d['d'] = 56
>>> d
{'b': 23, 'd': 56, 'a': 12, 'c': 34}
>>> d.pop('c')
34
>>> d
{'b': 23, 'd': 56, 'a': 12}
>>> 

===set=== 集合


>>> s = set('abc')
>>> s
{'b', 'a', 'c'}
>>> s = set([1, 2, 3])
>>> s
{1, 2, 3}
>>> s.add(4)
>>> s
{1, 2, 3, 4}
>>> s.add(3)
>>> s
{1, 2, 3, 4}
>>> s.remove(4)
>>> s
{1, 2, 3}
>>> 





===Python中的基本数据类型===

>>> type(5)                   //不变对象
<class 'int'>
>>> type(3.14)
<class 'float'>
>>> x = "Hello World..."                   //不变对象
>>> type(x)
<class 'str'>
>>> type(True)
<class 'bool'>
>>> type(None)                   //不变对象
<class 'NoneType'>
>>> 


二、条件

if-elif-[elif-]else

if <条件判断>:

if x:
    print('True');
只要x是非零数值、非空字符串、非空list等，就判断为True，否则为False。
